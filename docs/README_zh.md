## 1. 功能与特色

- **核心定位**: 最棒的浏览器网页总结工具！极速。极简。没有乱七八糟的AI杂烩功能。相信我！
- ⚪ **简单纯粹**: 简单、轻量、没有登录、没有远程后台服务、隐私优先（无遥测或数据收集）。
- ⚙ **配置选项**:
  - 🔧 自定义 LLM API（自带密钥）
  - 📝 定制 Prompt 模板
  - 📏 限制上下文长度
  - 🎛️ 多种触发方式：自动打开 / 悬浮球 / 右键菜单 / 快捷键 / 图标点击
  - ⚡ 自动开始总结 (启用/禁用)
  - 🛠️ 站点自定义 (Glob, 选择器, 白名单/黑名单)
- 🌊 **其他特性**:
  - 👀 查看 Token 使用情况
  - 🌐 适用于不同的站点权限设置

## 3. 核心定位

- 在浏览器中最方便快捷的网页内容总结工具
- 浏览器内置 AI 工具的纯净、轻量级替代品
- 不会实现**各种接入 + agent loop**功能，保持纯粹简单，*不再对标 sider.ai、monica.ai 等复杂产品*

*启发自: [chatGPTBox](https://github.com/josStorer/chatGPTBox)*
